# Banking Analytics Dashboard

## Overview
This project is an interactive Banking Analytics Dashboard developed using Power BI. It provides insights into customer demographics, transactions, loans, and credit risk.

## Dashboard Pages

### Executive Overview
- Total Customers
- Total Deposits
- Total Loans
- Total Withdrawals
- NPA %
- Deposit vs Withdrawal
- Monthly Transaction Trend
- Loan Status Distribution

### Customer Analytics
- Customer Demographics
- Income Segments
- Credit Score Distribution
- Gender Distribution
- Customer Distribution by City

### Transaction Analytics
- Total Transactions
- Channel Usage
- Monthly Deposit Trend
- Monthly Withdrawal Trend
- Top Branches

### Loan Analytics
- Loan Portfolio
- Average Loan Amount
- Average EMI
- Loan Status
- Top Loan Customers

### Risk Analytics
- NPA Analysis
- High Risk Customers
- Previous Defaults
- Credit Score Distribution
- Default Analysis

## Tools Used

- Power BI
- Power Query
- DAX
- Excel

## Features

- Interactive slicers
- KPI Cards
- Drill-down visuals
- Professional dashboard design
